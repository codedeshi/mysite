Surveys

  • AngularJS
  • Angular-Routes
  • Angular-Cookies
  • ExpressJS
  • NodeJS
  • MongoDB

A simple webpage that allows user to log and create poll questions. User can perform all CRUD (Create, Retrieve, Update and Delete) operation for the polls they created. The dashboard shows survey/poll created by all users and lets only the creator of the poll delete it. Validation is done before a new poll question is created. Any user can view any particular survey and vote any given option. These votes are updated in real time and user, survey question and votes are all saved in the database.

Bucket List

  • AngularJS
  • Angular-Routes
  • Angular-Cookies
  • ExpressJS
  • NodeJS
  • MongoDB

This webpage allows a user to log in and create a bucket list of activities they would like to do. They can also tag other users and the item gets added to their bucket list too. A users page will display there bucket list and also indicate which items are done and which are pending. A user can update their bucket list items to done by clicking on a check box next to the item. User can also visit other users profile and check their bucket lists however cannot update them. The check box status of each bucket list item is stored in the database.
