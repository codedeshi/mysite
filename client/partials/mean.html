Surveys

  • AngularJS
  • Angular-Routes
  • Angular-Cookies
  • ExpressJS
  • NodeJS
  • MongoDB

A fully integrated website created using MEAN. User can register/login and create survey questions, view survey questions created by others, and vote for any of the options provided. User can perform all CRUD (Create, Retrieve, Update and Delete) operations on any survey created by them. The votes are updated in real time and users, survey questions, and votes are all saved in the database.

Bucket List

  • AngularJS
  • Angular-Routes
  • Angular-Cookies
  • ExpressJS
  • NodeJS
  • MongoDB

This webpage allows a user to log in and create a bucket list of activities they would like to do. They can also tag other users and the item gets added to their bucket list too. User profile displays bucket list items in two categories, done and pending. Status of each bucket list item is stored in the database. User can update the status of any bucket list item by clicking on a check box next to the item. User can also visit other users profile and check their bucket lists however cannot update them.
